This is a sample blog post to show you how the blog system works. You can write your posts using plain HTML — just copy this file, rename it, and start writing. It's that simple.

How to Write a New Blog Post

Follow these steps to publish a new blog:

  1. Copy this file — Duplicate sample-post.html and rename it (e.g., my-new-post.html)
  2. Update the title — Change the <title> tag and the <h1>
  3. Update the metadata — Set the date, reading time, and tags
  4. Write your content — Replace this section with your blog content
  5. Add it to the listing — Add a card entry in blogs.html

Code Blocks

You can include code with syntax highlighting:

import torch
import torch.nn as nn

class SimpleNet(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, output_dim)
        )
    
    def forward(self, x):
        return self.net(x)
